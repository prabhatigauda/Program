Introduction

C is a procedural programming language it means it follows step by step process. It was initially developed by Dennis MacAlistair Ritchie to develop the UNIX operating system, At AT&T Bell laboratories of USA in 1972. During the 1980s C gradually gained popularity. Nowadays, it is one of the most widely used programming languages. C has many powers like it is simple, reliable, stretchy, portable and easy to use.

C and C++

While both C and C++ may sound similar, their features and usage are different. C is a procedural programming language and does not support objects and classes. C++ is an enhanced version of C programming with object-oriented programming support.

Some of the similarities in C and C++ programming languages are:

  • Both have a similar syntax.
  • Their compilation is similar.
  • Both languages follow the same concept of a stack, heap, and static variable.

Variables

An entity that may vary during program execution is called a variable. Variable names are names which are given to the locations in memory. These locations can contain integer, character constants.

Rules for constructing variable names of all types are same. These rules are as follows -
  • The first character in the variable name must be an alphabet or underscore.
  • No commas or blanks are allowed within a variable name.
  • No special symbol can be used in a variable except underscore.
  • A variable name is any combination of 1 to 31 alphabets, digits or underscores or we can say the maximum allowable length of a variable name is 31.
Constants

A Constant is an entity that doesn't change during the execution of a program.

Types of constants

Primary Constants

  • Real and Floating point Constant.
  • Integer Constant.
  • Character Constant.

Secondary Constants

  • Array
  • Pointer
  • Structure
  • enum
  • union

Details of these constants

Real and Floating Point constants

Real constant must have one digit in it and must have a decimal point either it is positive or negative no commas and blanks are allowed in real constants.

Integer Constants

An integer constant refers to a sequence of digits and it must have at least one digit either positive or negative, if no sign precedes an integer constant is assumed to be positive. commas and blanks are not allowed in integer constant. -32768 to 32768 is the range allowed for integer constant.

Character constants

Character constant is a single alphabet and a single digit enclosed within single inverted commas. Both inverted commas should point to the left.
Data Types

Each variable in C has an associated data type. Each data type requires different amounts of memory and has some specific operations which can be performed over it. Let us briefly describe them one by one:

Following are the examples of some very common data types used in C:
  • char: The most basic data type in C. It stores a single character and requires a single byte of memory in almost all compilers.
  • int: As the name suggests, an int variable is used to store an integer.
  • float: It is used to store decimal numbers (numbers with floating point value) with single precision.
  • double: It is used to store decimal numbers (numbers with floating point value) with double precision.
If Else Statement

If else statements in C is also used to control the program flow based on some condition, only the difference is: it's used to execute some statement code block if the expression is evaluated to true, otherwise executes else statement code block.

The basic format of if else statement is:

Syntex :

if(test_expression) { //execute your code } else { //execute your code }

Example :

#include void main() { int a, b; printf("Please enter the value for a:"); scanf("%d", &a); printf("\nPlease enter the value for b:"); scanf("%d", &b); if (a > b) { printf("\n a is greater"); } else { printf("\n b is greater"); } }

Output:

Please enter the value for a:7 Please enter the value for b:5 a is greater_
While Statement

while loop is a most basic loop in C programming. while loop has one control condition, and executes as long the condition is true.

The basic format of while loop statement is:

Syntex :

While (condition) { statement(s); Incrementation; }

Example :

#include int main () { /* local variable Initialization */ int n = 1,times=5; /* while loops execution */ while( n <= times ) { printf("C while loops: %d\n", n); n++; } return 0; }

Output :

C while loops: 1 C while loops: 2 C while loops: 3 C while loops: 4 C while loops: 5
